## Advanced layouts and images
# Element
Images, Margins, and Text

# Syntax
For images:
  <Image source={require('./my-icon.png')} />

For margins:
  /* Apply to all four sides */
  margin: 1em;
  margin: -3px;

  /* vertical | horizontal */
  margin: 5% auto;

  /* top | horizontal | bottom */
  margin: 1em auto 2em; 

  /* top | right | bottom | left */
  margin: 2px 1em 0 auto;

  /* Global values */
  margin: inherit;
  margin: initial;
  margin: unset;
